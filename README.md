# Qpyt-UI

Advanced modular interface for generative AI, powered by Diffusers, Florence-2 and Qwen.

<img width="2207" height="1017" alt="image" src="https://github.com/user-attachments/assets/2b50c7b9-c587-4594-86f9-0eeb4b2bfff2" />


## Key Features
- **Modular Brick System**: Drag and drop UI units for complex workflows.
- **Engines**: Support for SDXL, FLUX, and SD 3.5 Turbo.
- **Tiled Upscaler**: High-resolution super-sampling (up to 4K) via tiled diffusion.
- **Intelligent Routing**: Auto-pick last generated image for upscaling/refining.
- **Multi-tasking**: Prompt expansion (LLM), translation (MarianMT), and image analysis (Florence-2) integrated.

## Getting Started
Run `start.bat` to launch the FastAPI server and the web interface.
